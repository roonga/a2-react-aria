Two components: `RadioGroup` (the container) and `Radio` (individual options). Built on
React Aria's `RadioGroup` and `Radio` primitives with full keyboard and screen reader support.

## Add via CLI

```bash
npx @a2ra/cli add radio
```

## Basic

<ComponentPreview client:only="react" node={{
  type: "RadioGroup",
  props: {
    label: "Subscription plan",
    name: "plan",
    defaultValue: "monthly"
  },
  children: [
    { type: "Radio", props: { value: "monthly", label: "Monthly" } },
    { type: "Radio", props: { value: "annual", label: "Annual" } },
    { type: "Radio", props: { value: "lifetime", label: "Lifetime" } },
  ]
}} />

## RadioGroup props

| Prop | Type | Description |
|------|------|-------------|
| `label` | `string` | Group label |
| `value` | `string` | Controlled selected value |
| `defaultValue` | `string` | Uncontrolled default |
| `orientation` | `horizontal`<br />`vertical` <span className="prop-default-badge">default</span> | Layout direction |
| `isDisabled` | `true`<br />`false` <span className="prop-default-badge">default</span> | Disables all radios |
| `isRequired` | `true`<br />`false` <span className="prop-default-badge">default</span> | Marks group as required |
| `isReadOnly` | `true`<br />`false` <span className="prop-default-badge">default</span> | Prevents selection changes |
| `isInvalid` | `true`<br />`false` <span className="prop-default-badge">default</span> | Shows error styling |
| `validationBehavior` | `aria` <span className="prop-default-badge">default</span><br />`native` | Validation strategy |
| `name` | `string` | Form field name |
| `description` | `string` | Helper text |
| `errorMessage` | `string` | Error text when `isInvalid` is true |
| `children` | `node[]` | `Radio` child nodes |

## Radio props

| Prop | Type | Description |
|------|------|-------------|
| `value` | `string` | The value submitted when this radio is selected (required) |
| `label` | `string` | Visible label text |
| `isDisabled` | `true`<br />`false` <span className="prop-default-badge">default</span> | Disables just this option |

## Notes

- `Radio.value` is required. The `RadioGroup` uses it to track and submit the selection.
- Arrow keys move focus between options; `Space` selects the focused option.
- Only one `Radio` per group can be selected at a time.
