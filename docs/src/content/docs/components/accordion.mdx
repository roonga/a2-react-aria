`Accordion` and `AccordionItem` are built on React Aria's `DisclosureGroup` and `Disclosure`
primitives. Each item has a trigger button and a collapsible panel.

## Add via CLI

```bash
npx @a2ra/cli add accordion
```

## Basic

<ComponentPreview client:only="react" node={{
  type: "Accordion",
  children: [
    { type: "AccordionItem", props: { id: "q1", heading: "What is A2UI?", defaultExpanded: true }, children: "A2UI is a JSON schema for describing UI components that agents can emit." },

    { type: "AccordionItem", props: { id: "q3", heading: "Can multiple panels be open?" }, children: "Set allowsMultipleExpanded: true on the Accordion to allow it." },

}} />

## Example JSON

```json
{
  "type": "Accordion",
  "children": [
    {
      "type": "AccordionItem",
      "props": { "id": "q1", "heading": "What is A2UI?" },
      "children": "A2UI is a JSON schema for describing UI components."
    }
  ]
}
```

## Accordion props

| Prop | Type | Description |
|------|------|-------------|
| `allowsMultipleExpanded` | `true`<br />`false` <span className="prop-default-badge">default</span> | When true, multiple items can be open at once |
| `isDisabled` | `true`<br />`false` <span className="prop-default-badge">default</span> | Disables all items in the group |
| `children` | `node[]` | One or more `AccordionItem` nodes |

## AccordionItem props

| Prop | Type | Description |
|------|------|-------------|
| `heading` | `string` | Trigger button label (required) |
| `id` | `string` | Stable identifier for the disclosure |
| `defaultExpanded` | `true`<br />`false` <span className="prop-default-badge">default</span> | Whether the item starts open |
| `isDisabled` | `true`<br />`false` <span className="prop-default-badge">default</span> | Disables this item's trigger |
| `children` | `string`<br />`node[]` | Panel content |

## Notes

- `heading` is required on every `AccordionItem`; it becomes the accessible trigger button label.
- By default only one item can be open at a time. Opening a new item automatically closes the previous one.
- The chevron rotates 180° when the panel is expanded via a CSS transition on `[data-expanded]`.
