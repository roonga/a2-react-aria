A wrapper around React Aria's `DialogTrigger` and `Popover` components. Unlike `Tooltip`,
a `Popover` can contain interactive elements such as forms, links, or buttons.

## Add via CLI

```bash
npx @a2ra/cli add popover
```

## Basic

<ComponentPreview client:only="react" node={{
  type: "Popover",
  props: { triggerLabel: "Filter options", placement: "bottom" },
  children: [
    {
      type: "Form",
      props: { gap: "sm" },
      children: [
        { type: "Checkbox", props: { label: "In stock only", value: "instock" } },
        { type: "Select", props: {
          label: "Sort by",
          items: [
            { label: "Price: low to high", value: "price_asc" },
            { label: "Price: high to low", value: "price_desc" },
          ]
        }}
      ]
    }
  ]
}} />

## Example JSON

```json
{
  "type": "Popover",
  "props": {
    "triggerLabel": "Filter options",
    "placement": "bottom"
  },
  "children": [
    {
      "type": "Form",
      "props": { "gap": "sm" },
      "children": [
        { "type": "Checkbox", "props": { "label": "In stock only", "value": "instock" } }
      ]
    }
  ]
}
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `triggerLabel` | `string` | Label for the trigger button |
| `placement` | `top`<br />`bottom` <span className="prop-default-badge">default</span><br />`left`<br />`right` | Preferred position |
| `isOpen` | `boolean` | Controlled open state |
| `defaultOpen` | `true`<br />`false` <span className="prop-default-badge">default</span> | Open on mount |
| `offset` | `number` | Distance from the trigger in pixels. Default: `8` |
| `crossOffset` | `number` | Perpendicular offset in pixels. Default: `0` |
| `shouldFlip` | `true` <span className="prop-default-badge">default</span><br />`false` | Flip to opposite side when out of bounds |
| `isKeyboardDismissDisabled` | `true`<br />`false` <span className="prop-default-badge">default</span> | Prevent `Escape` from closing |
| `maxHeight` | `number` | Maximum height in pixels before scrolling |
| `children` | `node[]` | Content rendered inside the popover |

## Notes

- Use `Tooltip` for non-interactive hints. Use `Popover` when the content has clickable elements.
- `Escape` closes the popover and returns focus to the trigger by default.
- `maxHeight` enables internal scrolling for long content without affecting the page layout.
