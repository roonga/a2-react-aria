Two layout primitives: `Flex` for one-dimensional layouts (row or column) and `Grid` for
two-dimensional column grids. Both render as plain `<div>` elements using design-token spacing.

## Add via CLI

```bash
npx @a2ra/cli add layout
```

## Example JSON

```json
{
  "type": "Grid",
  "props": { "columns": 3, "gap": "md" },
  "children": [
    { "type": "Card", "children": [{ "type": "Text", "children": "Item 1" }] },
    { "type": "Card", "children": [{ "type": "Text", "children": "Item 2" }] },
    { "type": "Card", "children": [{ "type": "Text", "children": "Item 3" }] }
  ]
}
```

## Flex props

| Prop | Type | Description |
|------|------|-------------|
| `direction` | `row` <span className="prop-default-badge">default</span><br />`column`<br />`row-reverse`<br />`column-reverse` | Flex direction |
| `gap` | `none`<br />`xs`<br />`sm`<br />`md` <span className="prop-default-badge">default</span><br />`lg`<br />`xl` | Gap between children |
| `align` | `start` <span className="prop-default-badge">default</span><br />`center`<br />`end`<br />`stretch`<br />`baseline` | Cross-axis alignment |
| `justify` | `start` <span className="prop-default-badge">default</span><br />`center`<br />`end`<br />`between`<br />`around`<br />`evenly` | Main-axis distribution |
| `wrap` | `true`<br />`false` <span className="prop-default-badge">default</span> | Allow children to wrap |
| `children` | `node[]` | Child nodes |

## Grid props

| Prop | Type | Description |
|------|------|-------------|
| `columns` | `number` (1-12) | Number of equal-width columns. Default: `2` |
| `gap` | `none`<br />`xs`<br />`sm`<br />`md` <span className="prop-default-badge">default</span><br />`lg`<br />`xl` | Gap between cells |
| `align` | `start` <span className="prop-default-badge">default</span><br />`center`<br />`end`<br />`stretch` | Row alignment |
| `children` | `node[]` | Child nodes |

## Notes

- `Flex` and `Grid` render as plain `<div>` elements with no semantic meaning.
- For form layouts, prefer `Form` over `Flex` — `Form` wires up React Aria validation context.
- `Grid` columns are always equal-width. For asymmetric layouts, nest multiple `Flex` nodes.
