## How it works

**a2UI** is the JSON schema that AI agents emit. **a2ra** (a2 React Aria) is the renderer that
consumes it, validating every node with Zod and resolving it to the matching React Aria component in your project.

```text
AI Agent  →  { "type": "Button", "props": { "variant": "primary", "children": "Submit" } }
                    ↓  a2ra validates (Zod) + resolves (registry)
              <Button variant="primary">Submit</Button>   ← React Aria, accessible by default
```
